# LexiVault

**LexiVault** is a secure, local-first judicial intelligence platform designed for the Belize Senior Courts.

> **Zero-Cloud Policy**: Sensitive judicial data never leaves the local environment. All AI reasoning is performed on-premise using Ollama.

## 🚀 Quick Start

1.  **Prerequisites**: Docker, Node.js 20+, Python 3.9+, Ollama.
2.  **Setup**:
    ```bash
    cp .env.example .env
    npm install
    pip install -r requirements.txt
    ```
3.  **Run**:
    ```bash
    npm run dev
    ```
