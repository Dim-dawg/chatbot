# Product Roadmap

## Phase 1: Foundation (Current)
- [x] **Core Architecture**: Node.js + React + MySQL ecosystem.
- [x] **Zero-Cloud AI**: Local Ollama pipeline for embeddings and reasoning.
- [x] **Basic Dashboard**: Case summaries and lists.

## Phase 2: Operations (Q2 2026)
- [ ] **Accounting Module**: Cashiering and invoicing for court fees.
- [ ] **Vault Management**: Physical file tracking integration.
- [ ] **Advanced Analytics**: Judge workload balancing reports.

## Phase 3: Expansion (Q3 2026)
- [ ] **Public Access**: Restricted portal for attorneys and public inquiries.
- [ ] **Multi-Division**: Support for Civil, Criminal, and Family divisions.
- [ ] **Disaster Recovery**: Automated off-site encrypted backups.
