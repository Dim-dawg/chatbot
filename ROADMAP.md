# Product Roadmap

## Quarter 4 2024 (Current)
- [x] **Unified Command Center**: Dashboard and Chat on one screen.
- [x] **Global Case Search**: Instant access to 21,000+ files.
- [x] **Actionable AI**: Conversations can trigger database changes.
- [ ] **PDF Semantic Search**: Full implementation of PDF text extraction and vector search via Mistral.

## Quarter 1 2025
- [ ] **Authentication**: Implement secure judge/staff login.
- [ ] **Advanced Analytics**: Interactive charts for judge performance and court efficiency.
- [ ] **Document Summarization**: Ability for the AI to read a full PDF case file and provide a one-paragraph summary.

## Quarter 2 2025
- [ ] **Multi-Court Support**: Expanding the system to handle data from various court divisions.
- [ ] **Offline Knowledge Base**: Integrating legal statutes and local laws into the FAISS index.
